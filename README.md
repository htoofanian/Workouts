# Workouts
Gym Workout tracker - HTML file - Designed for phones, but also work on computers and iPads.  On iphone or ipad, open the link below in a browser and save it to your home page.

The dropdown below the heaading selects between (i) a push/pull/leg days routine, and (ii) an arms/legs/core days routine with alternating exercises. The button on the left of each exercises shows whether the exercise was done that week.  The button on the right indicates difficulty, or whether the weight or reps should be changed.  An exercise may be grouped together with a similar exercise below it. Grouped exercies have a blue border.  When grouped, doing either exercise means that group is completed. The default rep is 12x3 if not indicated. The Clear button clears all done exercises, but leaves in place the last difficulty level recorded.

These exercises can be edited, added, deleted or reordered using the Edit button. Exercise groups can be added or removed.  Day titles can also be edited, and additional days added (up to 7 days total). Edits are automatically saved locally to each device only, and will remain the next time the file/link is opened.  On a link saved to your home screen, these edits will not change the original source file linked to on github or any other location or device.  To save your edits to a revised  file that can be shared or used on other devices, select Save to Downloads, then move the downloaded file to your desired location.

Push/Pull app is based on doing all upper body push exercises one day, all leg exercises asecond day, and all upper body pull exercises a third day, with a rest day in between. Thisiss a muscle building routine.  

Maintain app is a little easier. It is based on alternating muscles, with a push or pull exercise usually followed an exercise for  the opposing muscle (e.g. biceps followed by triceps).  Upper body one day, legs a second day and core a third day.  This is a maintenance routine.
