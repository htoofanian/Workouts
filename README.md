# Workouts
Gym Workout trackers - HTML files - Designed for phones, but also work on computers and iPads.  On iphone or ipad, open the link below in a browser and save it to your home page.

The button on the right of each exercises shows whether the exercise was done.  The button on the right indicates difficulty, or whether the weight or reps should be increased or decreased.  The default rep is 12x3 if not indicated. Reset button clear all done exercises, but does not erase the last difficulty level recorded.

An exercise may be grouped together with a similar exercise below it.  Doing either exercise means that group is completed. 

These exercises can be edited, added, deleted or reordered. Exercise groups can be added or removed.  Day titles can also be edited, but there is always be 4 days/large categories. Edits are saved locally to each device only.  They will not change the linked file on any other device or the original file on github.
