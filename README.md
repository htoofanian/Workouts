# Workouts
Gym Workout trackers - HTML files - Designed for phones, but also work on computers and iPads.

(But you may need to update the version number to the current version.)

Workout app is based a weekly Push/Pull plan.  Push on one day, legs on a second day and pull on a third day, with rest days in between.

    https://htoofanian.github.io/Workouts/Workout-v32.html

Maintain app is a little easier. It is based on alternating muscles, with a push or pull exercise usually followed an exercise for  the opposing muscle (e.g. biceps followed by triceps).  Upper body one day, legs a second day and core a third day.  

    https://htoofanian.github.io/Workouts/Maintain-v49.html
